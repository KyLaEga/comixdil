# 📚 comixdil

Лёгкое десктоп-приложение для загрузки комиксов и манги практически с любого сайта со сборкой в **PDF** или **CBZ**.

## ✨ Возможности

- 🌐 **Универсальная загрузка** — на базе [`gallery-dl`](https://github.com/mikf/gallery-dl) со встроенными парсерами для `telegra.ph`, SexKomix и любой страницы с картинками комикса
- 📦 **Экспорт в PDF и CBZ** — сборка страниц в PDF (`img2pdf`) или CBZ-архив с регулировкой качества JPEG
- 🧠 **Экономия RAM** — страницы пишутся во временную папку на диск, а не держатся в памяти, поэтому большие галереи не съедают оперативку
- 🧵 **Многопоточность + rate-limit** — параллельные загрузки с настраиваемой задержкой и автоматическим замедлением при ошибках `503`
- ⏯️ **Управление очередью** — добавление множества ссылок сразу, пауза/возобновление, отмена, очистка завершённых
- 🎨 **Темы** — тёмная, светлая и системная на современном интерфейсе PySide6

## 🚀 Быстрый старт

```bash
git clone https://github.com/KyLaEga/comixdil.git
cd comixdil

pip install -r requirements.txt
python main.py
```

Нужен **Python 3.11+**. Зависимости (`requests`, `beautifulsoup4`, `Pillow`, `PySide6`, `gallery-dl`, `img2pdf`) ставятся из `requirements.txt`.

## 📋 Использование

1. Вставьте одну или несколько ссылок в поле ввода (**по одной на строку**).
2. Выберите формат (**PDF** или **CBZ**) и папку сохранения в сайдбаре.
3. Нажмите **«Скачать»** — программа найдёт картинки и соберёт файл.

**Совет:** настраивайте *Комиксов* (сколько комиксов параллельно), *Потоков* (потоков на один комикс) и *Задержку* в сайдбаре. Выше нагрузка — быстрее, но растёт риск блокировки `503`. Чтобы удалить задачи из очереди, выделите их и нажмите `Delete`.

## 🔧 Настройки (сайдбар)

| Параметр | Описание |
|----------|----------|
| **Папка** | Куда сохранять готовые файлы |
| **Формат** | `pdf` или `cbz` |
| **Качество** | Сжатие JPEG при сборке PDF |
| **Комиксов** | Сколько комиксов качать одновременно |
| **Потоков** | Потоков загрузки на один комикс |
| **Задержка (с)** | Пауза между запросами (защита от `503`) |
| **Тема** | Тёмная / Светлая / Системная |
